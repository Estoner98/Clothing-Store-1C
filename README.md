# Автоматизация магазина одежды (1С:Предприятие 8.3)

Проект по автоматизации учета в розничном магазине одежды. Разработан на платформе 1С:Предприятие 8.3.
Решение включает в себя рабочую конфигурацию, документацию для пользователей и сценарии тестирования.

## Структура репозитория

* **`ИС Магазин одежды.dt`** — Выгрузка информационной базы (основной файл проекта).
* **`Импорт Одежды.xlsx`** — Файл с данными для тестирования загрузки номенклатуры.
* **`/docs`** — Проектная документация (Руководство пользователя, User Stories, Словарь данных).
* **`/Диаграммы`** — Схемы бизнес-процессов.
* **`/Автоматизированное тестирование`** — Выгрузка с тестированием.

## Как запустить проект

Для ознакомления с разработкой вам понадобится установленная платформа **1С:Предприятие 8.3**.

1. **Скачайте файл** `ИС Магазин одежды.dt` из этого репозитория
2. Создайте чистую информационную базу в списке запуска 1С.
3. Откройте эту базу в режиме **Конфигуратор**.
4. В меню выберите: `Администрирование` -> `Загрузить информационную базу...`.
5. Выберите скачанный файл `.dt`.
6. После успешной загрузки запустите базу в режиме **1С:Предприятие**.

## Функциональность

* Учет поступления и продаж товаров (одежды).
* Хранение характеристик (размеры, цвета).
* Импорт данных из Excel (используйте файл `ИмпортОдежды.xlsx` для проверки).
* Формирование отчетов по остаткам и продажам.

## Примечание

* Отсутсвует Excel файл для импорта товаров.
